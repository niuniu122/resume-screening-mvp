# 模块日志 04：联调与验证模块

## 状态
- 已完成

## 目标
- 完成依赖安装、测试、构建、关键流程验证与最终运行说明。

## 安装命令
- 后端：`pip install -r backend/requirements.txt`
- 前端：`cd frontend && npm install`

## 测试命令与结果
- 后端：`cd backend && pytest -q`
- 结果：`2 passed`

## 构建命令与结果
- 前端：`cd frontend && npm run build`
- 结果：构建通过

## 已验证能力
- JD 文本导入可生成岗位与追问。
- 招聘方回答后可生成岗位画像草稿并冻结版本。
- 简历上传后可进入自动评估并在看板里查看候选人结果。
- 候选人报告可打开并执行人工改判。

## 风险与后续建议
- 当前环境默认走本地 SQLite 和本地文件存储，适合 MVP，不适合多机部署。
- 如果下一步要接真实大模型，建议补充模型输出 schema 校验与失败重试。
- 如果要面向正式招聘流程，建议继续补登录、权限、对象存储和任务队列。
